Это просто плагин, который позволяет отправлять уведомления в Telegram пользователям вашего сайта через службу Telenote.site.

== Description ==
Этот плагин для WooCommerce. На данный момент предлагается минимаьлная функциональность, но она будет расширяться со временем. Что делает наш плагин на данный момент:
* плагин добавляет кастомное поле на checkout странице где покупателю предлагается кликнуть по ссылке чтобы получать уведомления об изменении статуса заказа через месенджер Телеграм;
* плагин добавляет страницу в админке для настройки аккаунта сервиса Telenote и настройки самих уведомлений;
* плагин умеет отправлять уведомления клиенту и администратору о создании заказа и об изменнии статуса заказа.
* текст сообщений можно настроить с использованием различных плейсхолдеров.

== Frequently Asked Questions ==
= Отображается ли приглашение незарегистрированным пользователям? =
Нет, приглашение показывается только зарегистрировавшимся пользователям.
= Как мне не отправлять СМС, а только сообщение в Телеграм? =
Напишите нам на почту contact@telenote.site какие плагины (название и версия) вы используете для отправки СМС и мы добавим нужный код в наш плагин чтобы вы могли экономить.
